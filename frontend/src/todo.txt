DATA EXTRACTION PAGE
- To extract their data, the user has a menu with different checkboxes they can 
choose from, or all of them: their user profile, and any chat they participated 
in (DMs and the lobby). DMs are available even if the user hasn't posted 
anything in there yet, as it might be relevant to show the thread is empty, or 
to show what heinous things the so called friend has said. For the lobby 
though, they need to have posted at least one message.
- Let the user choose between JSON and CSV. Both are also possible.
- The user downloads the data from the website directly.
- Spinner for data extraction.

FRIENDSHIP & BLOCK
- Blocked list (private), link to users profile.
- From the blocked list, button to unblock.
- Friends list (public), link to users profile.
- From the friends list, button to remove friendship.
- Section for your pending requests. Button to cancel a request.
- Section for requests you have received. Button to accept and button to reject.
- The friends list is part of the public display, but without the button to 
remove the friendship of course.
