MODERATION
- A non-admin mod can renounce their rank.
- From the profile, a mod can remove the avatar, edit the username and the 
description.
- From the profile, the admin sees a button to give or remove the mod rank, as 
well as a button to delete the account.
- From chat, a mod can overwrite a user's specific chat message, while the 
admin can also overwrite a mod's message. The request doesn't specify a 
message, what it does is empty the message content, and add a boolean to say it 
has been moderated. On the frontend, the overwritten message is the same 
whether the mod is an ordinary mod or an admin: "Moderated message".
